Boite à onglets

Aide à créer une interface utilisateur organisée et facilitent la navigation de différents contenue sur une seule page dans un espace restreint.

Vue d'ensemble

Les onglets sont parfaits pour les applications Web d'une seule page. Ils permettent d’optimiser l’affichage du contenu dans un espace réduit grâce à un système d’étiquettes contrôlant l’affichage de panneaux relatifs.

Un onglet permet d'afficher dynamiquement et automatiquement le panneau associé et son contenu lorsque l'onglet reçoit le focus. Un seul onglet peut être activé à la fois.

Ce composant est totalement responsive et accessible. Il suit les recommandations wai-aria W3C Référence

Afin d'augmenter l'expérience utilisateur et notamment les utilisateurs de lecteurs d'écran par exemple. Un script Javascript est indispensable pour la gestion des onglets et l'affichage du contenu associé.

Ce script va permette d'uniformiser la hauteur des panneaux en ajustant la surface d'affichage au plus grand. Et de gérer la navigation dans les onglets, via les touches clavier.

La hauteur maximum des panneaux, peut être limitée grâce à la variable utilitaire --mh. Si le contenu est plus grand, alors une barre de défilement apparaitra automatiquement.
